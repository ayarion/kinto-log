# Kinto-Log 🏋️

かわいく続ける、筋トレ記録アプリ。

パステル水色×白のやさしいUIで、ジムでも**最短2タップ**でトレーニングを記録できます。
ログイン不要・データは端末内に自動保存。PWA対応なのでiPhoneのホーム画面に追加すればアプリのように使えます。

## 主な機能

- **記録** — 部位を選ぶと前回のメニューを自動で呼び出し。「そのまま記録」ならワンタップで保存
- **履歴** — 鍛えた部位が色ドットで見えるカレンダー。日付から過去の記録や写真を振り返り
- **からだ** — 体重・体脂肪率・筋肉量を記録、BMIは自動計算。からだの写真も残せる
- **グラフ** — AIコーチが「前回より+2.5kg」など客観的なひとことで背中を押す成長グラフ(期間切替つき)
- **バックアップ** — 記録をJSONで書き出し/読み込み

## 技術

- React 18 + Vite
- Recharts（グラフ）
- localStorage（端末内保存・ログイン不要）
- PWA（manifest + Service Worker・オフライン対応）

## 開発

```bash
npm install
npm run dev    # 開発サーバー (localhost:5173)
npm run build  # 本番ビルド (dist/)
```
